## 修复说明

**问题根因**：树叶生成条件 `depth >= maxDepth - 3` 导致高级树的分支太深才有叶子

对于10级树：
- `maxDepth = 12`，只有 `depth >= 9`（最后3层）才有叶子
- 这导致高级树反而叶子非常少

**修复方案**：改用深度比例 `depthRatio = depth / maxDepth`
- `depthRatio >= 0.4`（40%深度）就开始生成叶子
- 越深叶子概率越高：`leafProbability = 0.3 + depthRatio * 0.7`
- 这样无论树等级高低，都能有茂密的叶子
